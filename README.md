# Курс по дата-аналитике

Добро пожаловать в репозиторий курса по дата-аналитике! Здесь вы найдете все материалы, необходимые для успешного освоения курса, включая материалы лекций и домашние задания.

## Структура репозитория

- **inferential_statistics**: основы статистического анализа, работа с распределениями, проверка гипотез и многое другое.
- **git**: управление версиями, работа с репозиториями, ветвление и слияние.
- **SQL**: написание запросов, работа с базами данных, агрегация и фильтрация данных.
- **plotly**: создание интерактивных графиков и диаграмм.
- **streamlit**: создание веб-приложений для визуализации и анализа данных.
